## Installation
clone this repository
```javascript
git clone https://github.com/Book-Meet/YomiHomies.git
```
install dependencies
```javascript
npm install
```
install the expo cli
```javascript
npm install --global expo-cli
```
install the expo go app on your smartphone for testing.
